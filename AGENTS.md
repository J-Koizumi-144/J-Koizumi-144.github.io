# AGENTS.md

This repository contains a static GitHub Pages personal website.

## Site Overview

- The main entry page is `index.html`.
- Shared site styles live in `style.css`.
- Article-specific styles live in `articles/style.css`.
- The custom domain is configured in `CNAME`; do not change it unless explicitly asked.
- The site is mostly plain HTML/CSS with MathJax loaded where needed.

## Editing Rules

- Keep the site as static HTML/CSS unless the task explicitly asks for a build system or framework.
- Preserve the existing academic and mathematical style.
- Prefer small, focused edits over broad rewrites.
- Reuse existing CSS classes and page structure where practical.
- Keep MathJax-compatible TeX syntax for mathematical expressions.
- Do not modify PDFs, fonts, images, game assets, audio files, WebAssembly files, or generated Godot export files unless the task requires it.
- Do not remove analytics, `CNAME`, or existing navigation links unless explicitly requested.
- Use relative links that work on GitHub Pages.

## Verification

- For simple HTML/CSS edits, open the changed page locally in a browser when possible.
- Check that links, images, mathematical notation, and responsive layout still work.
- Before finishing, summarize the changed files and any verification performed.
